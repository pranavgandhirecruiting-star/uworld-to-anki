# UWorld → Anki

Anki add-on that lets you paste missed UWorld QIDs and instantly find + unsuspend the matching AnKing flashcards. No configuration required.

## Install

1. Download `uworld-to-anki.ankiaddon` from [Releases](../../releases)
2. Open Anki
3. Go to **Tools → Add-ons → Install from file** → select the downloaded file
4. Restart Anki
5. Go to **Tools → UWorld → Anki** — your browser opens and you're ready to go

That's it. No AnkiConnect, no CORS config, no terminal commands.

## How it works

The add-on embeds a lightweight web server inside Anki that:
- Serves a React UI in your browser
- Exposes API endpoints that talk directly to Anki's Python internals
- Runs on `127.0.0.1:28735` (localhost only, nothing leaves your machine)

No UWorld content is scraped, stored, or transmitted. The app only reads QID numbers you paste in and searches your local Anki deck for matching tags.

## Usage

1. Do a UWorld block
2. Filter for incorrect questions, copy the QIDs
3. Open **Tools → UWorld → Anki** (or go to `http://127.0.0.1:28735`)
4. Paste QIDs — any format works (comma-separated, one per line, with prefixes like `QID:`)
5. Click **Find Cards** — see all matching AnKing cards
6. Click **Unsuspend All** to activate them for review

## Development

```bash
# Install dependencies
npm install

# Start dev server (with hot reload)
# Requires the add-on running in Anki for API calls
npm run dev

# Build the React UI into the add-on
npm run build

# Package the .ankiaddon file
./build-addon.sh
```

The Vite dev server proxies `/api` requests to `127.0.0.1:28735` (the add-on server), so you can develop the UI with hot reload while Anki handles the API calls.

## Architecture

```
anki-addon/
  __init__.py      # Add-on entry point, API handlers, menu integration
  server.py        # Non-blocking HTTP server (socket + QTimer)
  manifest.json    # Anki add-on metadata
  web/             # Built React UI (generated by npm run build)

src/               # React + TypeScript source
  api/             # API client
  components/      # UI components
  hooks/           # React hooks
  utils/           # QID parser, session history
```
